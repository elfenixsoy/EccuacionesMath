Información sobre Ecuaciones

Ecuaciones de Primer Grado

Una ecuación de primer grado o ecuación lineal tiene la forma:

ax + b = 0

Donde a y b son constantes y a ≠ 0.

La solución se obtiene despejando x:

x = -b / a

Gráficamente, representa una línea recta que cruza el eje X en el punto solución.

Ecuaciones de Segundo Grado

Una ecuación de segundo grado o ecuación cuadrática tiene la forma:

ax² + bx + c = 0

Donde a, b y c son constantes y a ≠ 0.

Las soluciones se obtienen con la fórmula cuadrática:

x = [-b ± √(b² - 4ac)] / 2a

El discriminante Δ = b² - 4ac determina el tipo de raíces.

Tipos de Soluciones
Dos soluciones reales

Cuando Δ > 0

Una solución real

Cuando Δ = 0

Soluciones complejas

Cuando Δ < 0
